# Jumpman News Network

A fictional sports news hub for a multiplayer College Football 27 online dynasty. It includes headlines, a weekly spotlight matchup, power rankings, scores, and a Heisman watch.

## Update the dynasty

Edit the arrays at the top of `app.js` to change stories, rankings, scores, players, and the breaking-news ticker. Update the spotlight matchup directly in `index.html`.

The site is a dependency-free static project designed for GitHub Pages.
